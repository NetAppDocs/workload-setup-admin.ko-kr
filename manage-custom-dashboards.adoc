= 사용자 지정 대시보드 관리
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
대시보드 카탈로그에서 사용자 지정 대시보드를 관리하세요. 여기에서 세부 정보를 확인하고, 데이터를 새로 고치거나, 더 이상 필요하지 않은 대시보드를 삭제할 수 있습니다.

.이 작업에 대해
Workload Factory 계정의 모든 사용자는 모든 사용자 지정 대시보드에 액세스할 수 있습니다. 각 대시보드에는 생성자 이름 및 생성 날짜와 같은 기본 정보가 표시됩니다.

사용자 지정 대시보드는 읽기 전용입니다. 스토리지 또는 워크로드 설정을 변경할 수 없습니다.

.시작하기 전에
Workload Factory 계정에 최소 하나 이상의 사용자 지정 대시보드가 있어야 합니다.



[[access-the-dashboard-catalog]]
== 대시보드 카탈로그에 액세스합니다

.단계
. 다음 중 하나를 사용하여 로그인하세요link:console-experiences.html["콘솔 환경"^].
. 메뉴에서 *Workloads Overview*를 선택한 다음 *Custom dashboards*를 선택합니다.
+
대시보드 카탈로그에는 Workload Factory 계정의 모든 사용자 지정 대시보드가 표시됩니다.

. 대시보드 목록을 검토하세요.




[[refresh-dashboard-data]]
== 대시보드 데이터 새로 고침

대시보드를 열면 대시보드 데이터가 로드됩니다. 최신 정보를 보려면 새로고침하세요.

대시보드가 워크로드에서 데이터를 다시 로드합니다. 마지막 업데이트 시간이 변경되어 데이터가 새로 고쳐진 시간을 표시합니다.



[[delete-a-dashboard]]
== 대시보드 삭제

더 이상 필요하지 않은 대시보드를 삭제하세요. 이 작업은 영구적이며 되돌릴 수 없습니다.

.단계
. *사용자 지정 대시보드* 페이지에서 제거하려는 대시보드를 찾으세요.
. 해당 대시보드의 *작업* 드롭다운 메뉴에서 *삭제*를 선택합니다.
. 삭제를 확인하세요.


.결과
대시보드가 Workload Factory 계정에서 영구적으로 제거되며, 모든 사용자는 더 이상 대시보드에 접근할 수 없습니다.
